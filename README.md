# TCM-Food-Skill-System
中医时令养生食品开发系统 — 将传统中医智慧转化为合规、可商业化的现代健康食品。

📖 项目简介
本项目由 FormanGarden 研发团队 开发，旨在解决中医食疗在商业化过程中的三大痛点：医理转化难、合规风险高、成本波动大。

通过这套 Skill，任何配置了该能力的 AI Agent 都能变身为“中医食疗研发总监”，指导从配方论证到最终产品上架的全生命周期。

✨ V1.1 核心特性
时令对齐引擎 (Seasonal Alignment)：自动校准 24 节气，根据当前阴阳升降规律微调配方偏性。

商业成本预警 (Cost-Aware Logic)：针对酸枣仁等高波动原料建立方案 B 备选机制，保护商业利润。

全流程 SOP：涵盖医理论证、工艺设计、体验优化、合规防线及迭代验证五个阶段。
